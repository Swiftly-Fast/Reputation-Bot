Commands List

?rep {user}

Use this command to give a reputation point to another user. However, if you rep someone else later, your reputation point will be transferred to the new user. This ensures that at any given time, a user can only give one reputation point.

Usage Example: ?rep @User - Gives a reputation point to the mentioned user.

?reppoints {user}

This command lets you check how many reputation points a specific user has. If you don't specify a user, the bot will return your own reputation points in the current server. It's a quick way to see how appreciated you or others are in the community.

Usage Example: ?reppoints - Checks your own reputation points.
?reppoints @User - Checks the reputation points of the mentioned user.

?serverreps

Use this command to display a list of all users with reputation points in the current server, along with their point totals. This list is server-specific, so it will change depending on which server you are in when you run the command.

Usage Example: ?serverreps - Shows all users with reputation points in the current server.

?myreps

This command only works in Direct Messages (DMs). It gives you a detailed breakdown of how many reputation points you have across all servers where the bot is present. It's a great way to track your reputation activity and contributions across different communities.

Usage Example: ?myreps - Shows a summary of your reputation points across all servers, but only in DMs.

?globalrep {user}

This command displays the total reputation points of a specified user across all servers. If no user is specified, it will show your own total reputation points globally. This is useful for tracking how your reputation compares across multiple communities.

Usage Example: ?globalrep - Shows your total reputation points across all servers.
?globalrep @User - Shows the total reputation points for the mentioned user across all servers.

?repclear {user}

This command is used to clear the reputation points for a specified user in the current server. Only the bot owner can use this command. If no user is specified, it will clear all reputation points in the server.

Usage Example: ?repclear @User - Clears the reputation points for the mentioned user in the current server.
?repclear - Clears all reputation points in the server (owner only).

?repclearglobal {user}

This command clears all reputation points for a specified user across all servers. Like the ?repclear command, only the bot owner can use this command. If no user is specified, it will clear all reputation points for every user across all servers.

Usage Example: ?repclearglobal @User - Clears all reputation points for the mentioned user across all servers.
?repclearglobal - Clears all reputation points for every user across all servers (owner only).

/servers

This is a slash command that shows the number of servers the bot is currently in. It provides an invite link to add the bot to other servers. This command is helpful for users to understand the bot's reach and manage their own server.

Usage Example: /servers - Displays the number of servers the bot is in along with an invite link.

?commands

This command lists all the commands available for the bot, along with a brief description of each. It's useful for users to quickly familiarize themselves with the bot's functionalities and how to interact with it.

Usage Example: ?commands - Displays all commands and their descriptions.
